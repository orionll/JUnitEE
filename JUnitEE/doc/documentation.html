Available Documentation

We have a full set of documentation available to guide you through the use of JUnitEE.

For a quick start the JUnitEE Tutorial will give you a step-by-step description of how to set up and use JUnitEE.

The JUnitEE User's Guide contains a complete description of JUnitEE and also documents the more advanced features not mentioned in the JUnitEE Tutorial.

If you use Jakarta Ant as your build system and would like to integrate JUnitEE with Ant, please have a look at the Ant Integration Guide.
